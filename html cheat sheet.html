üìÑ Basics

  • <html> ... </html>: Defines an HTML document.
  • <head> ... </head>: Contains metadata, links, and title.
  • <body> ... </body>: Contains the visible content.
  • <p>: Defines a paragraph.
  • <h1> to <h6>: Headings of different sizes.
  • <div>: A container for structuring content.
  • <span>: Inline container for text and styles.

üìù Forms

  • <form>: Defines a form for user input.
  • <input>: Defines input fields (e.g., text, password).
  • <textarea>: Multi-line text input.
  • <button>: A clickable button.
  • <select>: Dropdown list with options.
  • <label>: Labels for form elements.
  • <fieldset> & <legend>: Group form elements together.

Example:


<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name" required><br>
    <button type="submit">Submit</button>
</form>

üé• Media

  • <img>: Adds an image. Attributes: src, alt.
  • <video>: Embeds a video. Attributes: controls, autoplay.
  • <audio>: Embeds an audio file. Attributes: controls, loop.
  • <figure> & <figcaption>: Groups media with a caption.

Example:


<figure>
    <img src="example.jpg" alt="A beautiful scenery">
    <figcaption>A beautiful scenery</figcaption>
</figure>

üìä Tables

  • <table>: Defines a table.
  • <tr>: Table row.
  • <th>: Table header cell.
  • <td>: Table data cell.
  • colspan, rowspan: Merge cells.

Example:


<table>
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

üí° Tips & Tricks

  • Use semantic tags like <article> and <section> for better SEO.
  • Always include the alt attribute in images for accessibility.
  • Use <template> for reusable HTML blocks.
  • Minimize inline styles; use CSS instead.
